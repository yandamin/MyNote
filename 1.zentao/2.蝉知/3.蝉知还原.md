
# pro1.9 > 7.5
```
1、备份蝉知的数据库；
2、把当前蝉知运行文件夹chanzhieps（以chanzhieps为例介绍操作步骤）改名为chanzhieps_pro19；
3、下载开源版7.5版本源码包： http://dl.cnezsoft.com/chanzhi/7.5/chanzhiEPS.7.5.zip ，解压后得到chanzhieps文件夹，将chanzhieps拷贝到和chanzhieps_pro19 相同的目录；
4、复制chanzhieps_pro19/www/data 整体拷贝到 chanzhieps/www/下面；
5、复制chanzhieps_pro19/system/tmp/template 整体拷贝到 chanzhieps/system/tmp/ ;
6、复制chanzhieps_pro19/system/config/my.php 拷贝到 chanzhieps/system/config/ ;
7、（如果安装主题执行此步骤，如果没有安装直接调过此步骤）复制 chanzhieps_pro19/system/module/ui/ext/config/下面的文件拷贝到 chanzhieps/system/module/ui/ext/config/ 下面；
8、（如果安装插件执行此步骤，如果没有安装直接调过此步骤）到 chanzhieps_pro19/system/tmp/package/ 下面找到相应的插件包（注意只是插件包，不要类似qd0050.zip开头的zip包），拷贝到chanzhieps/system/tmp/package/下面。另外解压一份插件包，把里面的文件覆盖到chanzhieps/下面对应的文件目录中;
9、linux系统可能还需要注意一下chanzhieps整个文件的读写权限，可以执行一下 chmod -R 777 chanzhieps
```
